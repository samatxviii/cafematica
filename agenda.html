Agenda Cafemática

Organize compromissos com estrutura simples.

Cadastre aulas, consultorias, gravações e tarefas. Cada compromisso recebe um nome de arquivo padronizado, como 2026-06-30-01.txt, pronto para exportação e organização.

Criar compromissoVer agenda

Como funciona

  • Novo compromisso: a agenda gera o próximo número disponível para a data.
  • Edição: o nome do arquivo permanece o mesmo.
  • Exportação: baixe compromissos em arquivos .txt com nomes organizados.
  • Importação: traga de volta arquivos .txt exportados anteriormente.
Cadastro

Novo compromisso

Preencha a data, o título e o corpo do texto. A numeração do arquivo será definida automaticamente.

Arquivo previsto: escolha uma data

Ao editar um compromisso, o nome do arquivo permanece o mesmo. Isso evita quebrar sua organização.

Padrão de arquivo

Nomes previsíveis.

A agenda segue a lógica: AAAA-MM-DD-NN.txt. O número final identifica a ordem de cadastro naquele dia.

2026-06-30-01.txt2026-06-30-02.txt2026-07-01-01.txt

O conteúdo do arquivo exportado inclui título, data, tipo, data de criação, data de atualização e corpo do texto.

Agenda

Compromissos cadastrados

Os dados abaixo estão salvos neste navegador. Use exportação/importação para manter cópias externas.

Comandos simples

Cadastre sem perder tempo.

Use comandos rápidos quando quiser registrar ou alterar algo de forma textual.

ADD 2026-06-30 | Título | Corpo do texto
EDIT 2026-06-30-01.txt | Novo título | Novo corpo do texto
DELETE 2026-06-30-01.txt

No comando EDIT, o arquivo mantém o mesmo nome. No comando ADD, a agenda calcula automaticamente o próximo número da data.

Exportação

Baixe seus .txt.

Exporte todos os compromissos em um único arquivo JSON ou baixe cada compromisso individualmente em TXT.

Por segurança do navegador, vários TXT serão baixados um por vez. Para muitos compromissos, o JSON é mais prático.

Importação

Traga arquivos de volta.

Importe arquivos TXT exportados pela agenda ou um arquivo JSON de backup.

Ao importar um arquivo com o mesmo nome de outro já existente, a agenda atualiza o compromisso correspondente.
