Compromissos cadastrados

Os dados abaixo estão salvos neste navegador. Use exportação/importação para manter cópias externas.

Comandos simples

Cadastre sem perder tempo.

Use comandos rápidos quando quiser registrar ou alterar algo de forma textual.

ADD 2026-06-30 | Título | Corpo do texto
ADD 2026-06-30 14:00-15:00 | Título | Corpo do texto
EDIT 2026-06-30-01.txt | Novo título | Novo corpo do texto
DELETE 2026-06-30-01.txt

No comando EDIT, o arquivo mantém o mesmo nome. No comando ADD, a agenda calcula automaticamente o próximo número da data.

Exportação

Baixe seus .txt.

Exporte todos os compromissos em um único arquivo JSON ou baixe cada compromisso individualmente em TXT.

Por segurança do navegador, vários TXT serão baixados um por vez. Para muitos compromissos, o JSON é mais prático.

Importação

Traga arquivos de volta.

Importe arquivos TXT exportados pela agenda ou um arquivo JSON de backup.

Ao importar um arquivo com o mesmo nome de outro já existente, a agenda atualiza o compromisso correspondente.
