Compromissos cadastrados

Os dados abaixo estão salvos neste navegador. Use exportação/importação para manter cópias externas.

Exportação

Baixe seus compromissos.

Exporte todos os compromissos em um único arquivo JSON ou baixe cada compromisso individualmente em TXT.

Para muitos compromissos, o JSON é mais prático, para impressão gere o PDF.

Backup prático para celular

Copie a agenda como texto e guarde em uma conversa, nota, e-mail ou bloco de notas. Depois é só colar de volta na importação.

Importação

Carregue seus compromissos.

Importe arquivos TXT ou JSON exportados pela agenda.

Ao importar a agenda apenas são inseridos os novos compromissos.

Importar backup colado

Cole aqui o texto copiado da agenda. Funciona melhor no celular do que procurar um arquivo JSON na pasta de downloads.
